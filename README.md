# 🌍 Capital Finder (with Quiz Game)

**Capital Finder** — це сучасний, адаптивний вебзастосунок для пошуку інформації про країни та їхні столиці. Проєкт створено з використанням React та TypeScript, а дані завантажуються в реальному часі з публічного REST API. Також у застосунок інтегровано інтерактивну мінігру-квіз для перевірки знань.

## ✨ Особливості (Features)

* 🔍 **Миттєвий пошук:** пошук країни за назвою з автоматичною валідацією введення (тільки літери).
* 📱 **Повна адаптивність (Mobile First):** інтерфейс ідеально підлаштовується під будь-які екрани (від 320px до десктопів).
* 🎮 **Interactive Quiz:** вбудована мінігра "Вгадай столицю" з динамічним підрахунком балів та підсвічуванням відповідей.
* 📜 **Історія пошуку:** збереження останніх 5 запитів у `localStorage` для швидкого доступу в один клік.
* 🎨 **Плавні анімації:** мікроінтеракції, анімований заголовок та плавці CSS-ефекти випливання карток.
* 🛠️ **UX/UI покращення:** розумний інпут з кнопкою швидкого очищення (×), автофокусом та повним захистом від зайвих пробілів (`.trim()`).
* 🛑 **Обробка помилок (Error Handling):** гарні "Empty States" для порожньої історії та кастомна картка помилки, якщо країну не знайдено.

---

## 🛠️ Технологічний стек (Tech Stack)

* **Frontend:** React (Hooks: `useState`, `useEffect`, `useRef`)
* **Мова:** TypeScript
* **Стилізація:** CSS Modules (чистий CSS, Адаптивна верстка, CSS Keyframes)
* **Запити до API:** Axios
* **Дані:** [Rest Countries API](https://restcountries.com/)
* **Збірка:** Vite
